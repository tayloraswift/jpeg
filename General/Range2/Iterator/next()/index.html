Instance Method

next()

Advances to the next element and returns it, or nil if no next element exists.

Implements requirement in IteratorProtocol.

Declaration

mutating func next() -> (x:Bound, y:Bound)?

Return value

The next element in the two-dimensional range sequence, if it exists, otherwise nil. If advancing the x index would cause it to reach its upper bound, this iterator will advance to the next y index and reset the x index to its lower bound.
