Instance Method

makeIterator()

Creates an iterator for this range instance.

Implements requirement in Sequence. Available when Bound conforms to Strideable and Bound.Stride conforms to SignedInteger.

Declaration

func makeIterator() -> Iterator

Return value

An iterator.

Overview

This iterator will traverse the range space in row-major order. For example, if the bounds are (x: 0, y: 0) and (x: 2, y: 2), the iterator will yield the elements (x: 0, y: 0), (x: 1, y: 0), (x: 0, y: 1), and (x: 1, y: 1), in that order.
