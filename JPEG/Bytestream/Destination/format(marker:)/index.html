Instance Method

format(marker:)

Formats a single marker into this bytestream.

Declaration

mutating func format(marker:JPEG.Marker) throws

Parameters

marker

The type indicator of the marker to format.

Overview

This function is meant to be used with markers without segment bodies, such as start, end, and restart(_:).

This function can throw a FormattingError if it fails to write to the bytestream.
