Instance Method

format(marker:tail:)

Formats a single marker segment into this bytestream.

Declaration

mutating func format(marker:JPEG.Marker, tail:[UInt8]) throws

Parameters

marker

The type indicator of the marker to format.

tail

The marker segment body. This array should not include the marker type indicator, or the marker segment length field.

Overview

This function will output a segment length field, even if no marker data is provided, and so should not be used with markers without segment bodies, such as start, end, and restart(_:).

This function can throw a FormattingError if it fails to write to the bytestream.
