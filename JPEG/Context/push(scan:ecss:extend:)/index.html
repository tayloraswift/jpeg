Instance Method

push(scan:ecss:extend:)

Updates the decoder state with the given scan header, and decodes the given entropy-coded segment.

Declaration

mutating func push(scan:JPEG.Header.Scan, ecss:[[UInt8]], extend:Bool) throws

Parameters

scan

The scan header associated with the given entropy-coded segment.

ecss

The entropy-coded segment. Each sub-array is one restart interval of segment.

extend

Specifies whether or not the decoder is allowed to dynamically extend the height of the image if the entropy-coded segment contains more rows of image data than implied by the frame header size.

This argument should be set to true for the first scan in the file, to accommodate a possible HeightRedefinition, and false for all other scans.

Overview

This type tracks the scan progression of the stored image, and will validate the newly pushed scan header against the stored progressive state.
