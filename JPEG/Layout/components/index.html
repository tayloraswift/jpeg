Instance Property

components

A dictionary mapping all of the resident components in the image to their sampling factors and quanta keys.

Declaration

var components:[JPEG.Component.Key: (factor:(x:Int, y:Int), qi:JPEG.Table.Quantization.Key)] { get }

Overview

This property should be equivalent to the components dictionary that would be used to construct this instance using init(format:process:components:scans:). As long as the Format type is properly implemented, this dictionary will always have at least one element.
