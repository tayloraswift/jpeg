Instance Method

set(width:)

Sets the width of this image, in pixels.

Declaration

mutating func set(width x:Int)

Parameters

x

The new width of this image, in pixels. This width is measured from the left side of the image. Passing a negative or zero value will result in a precondition failure.

Overview

Existing data in this image will either be preserved, or cropped. Any additional data units created by this function will have all coefficients initialized to zero.
