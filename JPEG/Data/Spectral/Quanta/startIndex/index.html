startIndex

The index of the first quantization table in this container.

Implements requirement in RandomAccessCollection.

Declaration

var startIndex:Int { get }

Overview

The default (all-zeroes) quantization table is not part of the RandomAccessCollection. This index is 1 greater than the index of the default quanta.
