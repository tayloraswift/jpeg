Instance Method

index(forKey:)

Returns the index of the table with the given key.

Declaration

func index(forKey qi:JPEG.Table.Quantization.Key) -> Int

Parameters

qi

The quanta key. Passing a key that does not exist in this container will result in a precondition failure.

Return value

The integer index. This index can be used with the [_:] subscript.

Overview

An instance of this type which is part of a Spectral instance will always contain all quanta keys used by its layout, including keys used only by non-recognized components.
