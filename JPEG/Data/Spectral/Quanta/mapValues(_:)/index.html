mapValues(_:)

Returns a dictionary of the quantization tables in this container with the quantum values of each table transformed by the given closure.

Declaration

func mapValues<R>(_ transform:([UInt16]) throws -> [UInt16]) rethrows -> [JPEG.Table.Quantization.Key: R]

Parameters

transform

A closure that transforms a value. This closure accepts a 64-element zigzag-indexed array of the quantum values in each table as its parameter, and returns a transformed value of the same or of a different type.

Return value

A dictionary containing the keys and transformed quanta of the quantization tables in this container.
