Instance Method

compress(path:)

Compresses a spectral image to the given file path.

Declaration

func compress(path:String) throws -> Void?

Parameters

path

A file path.

Return value

A Void tuple, or nil if the file could not be opened at the given file path.

Overview

All metadata records in this image will be emitted at the beginning of the outputted file, in the order they appear in the metadata array.

Calling this function is equivalent to calling System.File.Destination.open(path:_:) with the closure parameter set to Self.compress(stream:).

This function is only available on MacOS and Linux platforms.
