Subscript

[x:y:k:h:]

Accesses the frequency coefficient at the specified grid index in the specified data unit.

Declaration

@inlinable subscript(x x:Int, y y:Int, k k:Int, h h:Int) -> Int16 { get set }

Parameters

x

The horizontal index of the data unit to access.

y

The vertical index of the data unit to access. Index 0 corresponds to the visual top of the image.

k

The horizontal frequency index of the coefficient to access. This value must be in the range 0 ..< 8.

h

The vertical frequency index of the coefficient to access. This value must be in the range 0 ..< 8.

Return value

The frequency coefficient.

Overview

The x and y indices of this subscript have no index bounds. Out-of-bounds reads will return 0; out-of-bounds writes will have no effect. The k and h indices still have to be within the correct range.

Using this subscript is equivalent to using [x:y:z:] with the z index set to the output of Table.Quantization.z(k:h:).
