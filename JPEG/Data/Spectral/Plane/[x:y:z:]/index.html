Subscript

[x:y:z:]

Accesses the frequency coefficient at the specified zigzag index in the specified data unit.

Declaration

subscript(x x:Int, y y:Int, z z:Int) -> Int16 { get set }

Parameters

x

The horizontal index of the data unit to access.

y

The vertical index of the data unit to access. Index 0 corresponds to the visual top of the image.

z

The zigzag index of the coefficient to access. This index must be in the range 0 ..< 64.

Return value

The frequency coefficient.

Overview

The x and y indices of this subscript have no index bounds. Out-of-bounds reads will return 0; out-of-bounds writes will have no effect. The z index still has to be within the correct range.
