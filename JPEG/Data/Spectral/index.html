Generic Structure

JPEG.Data.Spectral

A planar image represented in the frequency domain.

Declaration

struct Spectral<Format>:RandomAccessCollection where Format:JPEG.Format

Overview

A spectral image stores its data in blocks called data units. Each block is a square 8Ã—8 matrix of frequency coefficients. The data units themselves have the same spatial arrangement they do in the spatial domain.

A spectral image always stores a whole number of data units in both dimensions, even if the image dimensions in pixels are not multiples of 8. Because each component in an image has its own sampling factors, the image planes may not have the same size.

The spectral representation is a lossless representation. JPEG images that have been decoded to this representation can be re-encoded without loss of information or compression.
