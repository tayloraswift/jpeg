Initializer

init(size:layout:metadata:values:)

Creates a rectangular image with the given image parameters and interleaved samples.

Declaration

init(size:(x:Int, y:Int), layout:JPEG.Layout<Format>, metadata:[JPEG.Metadata], values:[UInt16])

Parameters

size

The size of the image, in pixels. Both dimensions must be positive.

layout

The layout of the image.

metadata

The metadata records in the image.

values

An array of interleaved samples, in row major order, and without padding. The array must have exactly layout.recognized.count × size.x × size.y samples. Each UInt16 is one sample. The samples should not be normalized, so an image with a layout.format.precision of 8 should only have samples in the range 0 ... 255.

Overview

Passing an invalid size, or an array of the wrong count will result in a precondition failure.
