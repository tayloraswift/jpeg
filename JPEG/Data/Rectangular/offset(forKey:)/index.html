Instance Method

offset(forKey:)

Returns the interleaved offset of the color channel represented by the given component key, or nil if the component key is a non-recognized component.

Declaration

func offset(forKey ci:JPEG.Component.Key) -> Int?

Parameters

ci

The component key.

Return value

The interleaved offset of the channel, or nil. If not nil, this offset can be used as the p parameter to the [x:y:p:] subscript.
