Generic Static Method

decompress(stream:cosite:)

Decompresses a rectangular image from the given data source.

Declaration

static func decompress<Source>(stream:inout Source, cosite cosited:Bool) throws -> Self where Source:JPEG.Bytestream.Source

Parameters

stream

A source bytestream.

cosited

The upsampling method to use. Setting this parameter to true co-sites the samples; setting it to false centers them instead.

The default value is false.

Return value

The decompressed image.

Overview

This function is a convenience function which calls Planar.decompress(stream:) to obtain a planar image, and then calls interleaved(cosite:) on the output to return a rectangular image.
