Instance Property

stride

The stride of the interleaved samples in this image.

Declaration

let stride:JPEG.Layout<Format>

Overview

This value is analogous to the plane count of a planar or spectral image. For example, the rectangular representation of a planar YCbCr image with 3 planes would have a stride of 3.
