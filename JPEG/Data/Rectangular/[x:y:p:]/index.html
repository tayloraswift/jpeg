Subscript

[x:y:p:]

Accesses the sample at the specified pixel location and offset.

Declaration

subscript(x x:Int, y y:Int, p p:Int) -> UInt16 { get set }

Parameters

x

The horizontal pixel index of the sample to access.

y

The vertical pixel index of the sample to access. Index 0 corresponds to the visual top of the image.

p

The interleaved offset of the sample. This offset is analogous to the plane index in the planar image representations.

Return value

The sample.
