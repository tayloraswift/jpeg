decompress(stream:)

Decompresses a planar image from the given data source.

Declaration

static func decompress<Source>(stream:inout Source) throws -> Self where Source:JPEG.Bytestream.Source

Parameters

stream

A source bytestream.

Return value

The decompressed image.

Overview

This function is a convenience function which calls Spectral.decompress(stream:) to obtain a spectral image, and then calls idct() on the output to return a planar image.
