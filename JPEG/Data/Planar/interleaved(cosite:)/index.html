Instance Method

interleaved(cosite:)

Converts this planar image into its rectangular representation.

Declaration

func interleaved(cosite cosited:Bool) -> JPEG.Data.Rectangular<Format>

Parameters

cosited

The upsampling method to use. Setting this parameter to true co-sites the samples; setting it to false centers them instead.

The default value is false.

Return value

A rectangular image created by upsampling all planes in the input to the same sampling factor.
