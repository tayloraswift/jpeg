Instance Property

bits

The bit range encoded by this scan.

Declaration

let bits:Range<Int>

Overview

This property specifies a range of bit indices, where bit zero is the least significant bit. The upper range bound is always either infinity (Int.max) or one greater than the lower bound. If the image coding process is not progressive(coding:differential:), this value will be 0 ..< .max.
