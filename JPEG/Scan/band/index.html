Instance Property

band

The frequency band encoded by this scan.

Declaration

let band:Range<Int>

Overview

This property specifies a range of zigzag-indexed frequency coefficients. It is always within the interval of 0 to 64. If the image coding process is not progressive(coding:differential:), this value will be 0 ..< 64.
