Instance Property

selector

The table selectors for the huffman tables associated with this component in the context of this scan.

Declaration

let selector:(dc:JPEG.Table.HuffmanDC.Selector, ac:JPEG.Table.HuffmanAC.Selector)

Overview

An image component may use different huffman tables in different scans. (In contrast, quantization table assignments are global to the file.) The DC table is used to encode or decode coefficient zero; the AC table is used for all other frequency coefficients. Depending on the band and bit range encoded by the image scan, one or both of the huffman table selectors may be unused, and therefore may not need to reference valid tables.
