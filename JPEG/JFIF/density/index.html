Instance Property

density

The sampling density of the image.

Declaration

let density:(x:Int, y:Int, unit:Unit?)

Overview

The x and y fields specify the number of physical samples along each dimension per unit length. For example, the value (x: 72, y: 72, unit: Unit.inches) indicates a density of 72Â dpi. If the unit field is nil, the units are unknown.
