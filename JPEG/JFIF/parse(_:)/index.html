parse(_:)

Parses a JFIF segment into a metadata record.

Declaration

static func parse(_ data:[UInt8]) throws -> Self

Parameters

data

The segment data to parse.

Return value

The parsed metadata record.

Overview

If the given data does not parse to a valid metadata record, this function will throw a ParsingError.

This parser ignores embedded thumbnails.
