Enumeration

JPEG.CompressionLevel

A basic image quality parameter.

Declaration

enum CompressionLevel

Overview

This is a toy API which generates acceptable defaults for a range of quality settings. For finer-grained control, specify coefficient-wise quantum values manually.

Topics

Enumeration cases

case chrominance(Double)

A quality level for a chrominance component.

case luminance(Double)

A quality level for a luminance component.

Instance properties

var quanta:[UInt16]

A 64-component array containing quantum values determined by this quality parameter, in zigzag order.
