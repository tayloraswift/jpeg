Static Method

validate(process:band:bits:components:)

Creates a scan header after validating the given field values.

Declaration

static func validate(process:JPEG.Process, band:Range<Int>, bits:Range<Int>, components:[JPEG.Scan.Component]) throws -> Self

Parameters

process

The coding process used by the image.

band

The frequency band encoded by the scan, in zigzag order. It must be within the interval of 0 to 64. If the process is progressive(coding:differential:), this parameter must either be 0 ..< 1, or some range within the interval 1 ..< 64. Otherwise, this parameter must be set to 0 ..< 64.

bits

The bit range encoded by the scan, where bit zero is the least significant bit. The upper range bound must be either infinity (Int.max) or one greater than the lower bound. If the process is not progressive(coding:differential:), this value must be set to 0 ..< .max.

components

The color components in the scan, in the order in which their data units are interleaved. If the scan is an AC progressive scan, this array must have exactly one element. Otherwise, it must have between one and four elements. If the process is baseline, the components can only use the huffman table selectors \.0 and \.1.

Return value

A scan header.

Overview

If the given parameters are not consistent with one another, and the JPEG standard, this function will throw a ParsingError.
