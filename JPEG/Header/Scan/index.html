Structure

JPEG.Header.Scan

A scan header.

Declaration

struct Scan

Overview

This structure is the parsed form of a scan marker segment. It defines scan-level image parameters. The library validates these structures against the global image parameters to create the JPEG.Scan structures elsewhere in the library API.
