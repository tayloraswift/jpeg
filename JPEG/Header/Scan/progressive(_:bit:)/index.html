Static Method

progressive(_:bit:)

Creates a progressive refining DC scan descriptor.

Declaration

static func progressive(_ components:[JPEG.Component.Key], bit:Int) -> Self

Parameters

components

The components encoded by this scan. Passing an empty array will result in a precondition failure.

bit

The index of the bit refined by this scan. Passing a negative index will result in a precondition failure.

Return value

An unvalidated refining DC scan header.

Overview

This constructor bypasses normal scan header validation checks. It is otherwise equivalent to calling validate(process:band:bits:components:) with the band argument set to 0 ..< 1 and the bits argument set to bit ..< bit + 1.

Refining DC scans do not use entropy-coding, so no huffman table selectors need to be specified.
