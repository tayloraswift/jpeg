Structure

JPEG.Header.RestartInterval

A restart interval definition header.

Declaration

struct RestartInterval

Overview

This structure is the parsed form of an interval marker segment. It can modify or clear the restart interval of an image.

Topics

Initializers

init(interval:Int?)

Creates a restart interval definition header.

Instance properties

let interval:Int?

The restart interval, in minimum-coded units, or nil if the header is meant to disable restart markers.

Type methods

static func parse([UInt8]) throws -> Self

Parses an interval segment into a restart interval definition.
