Enumeration Case

extended(coding:differential:)

The extended coding process.

Declaration

case extended(coding:Coding, differential:Bool)

Associated values

coding

The entropy coding used by this coding process.

differential

Indicates whether the image frame using this coding process is a differential frame under the hierarchical mode of operations.

Overview

This is a sequential coding process. It allows up to four simultaneously referenced tables of each type. It can only be used with color formats with a bit precision of 8 or 12.
