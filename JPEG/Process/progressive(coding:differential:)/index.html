Enumeration Case

progressive(coding:differential:)

The progressive JPEG coding process.

Declaration

case progressive(coding:Coding, differential:Bool)

Associated values

coding

The entropy coding used by this coding process.

differential

Indicates whether the image frame using this coding process is a differential frame under the hierarchical mode of operations.

Overview

This is a progressive coding process. It allows up to four simultaneously referenced tables of each type. It can only be used with color formats with a bit precision of 8 or 12, and no more than four components.
