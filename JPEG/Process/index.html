JPEG.Process

A coding process.

Declaration

enum Process

Overview

The JPEG standard specifies several subformats of the JPEG format known as coding processes. The library can recognize images using any coding process, but only supports encoding and decoding images using the baseline, extended(coding:differential:), or progressive(coding:differential:) processes with huffman entropy coding and the differential flag set to false.
