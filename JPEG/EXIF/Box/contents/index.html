Instance Property

contents

The contents of this field box.

Declaration

let contents:(UInt8, UInt8, UInt8, UInt8)

Overview

Depending on the field, the four bytes that make up this property may encode the field value directly, or encode a 32-bit internal file address pointing to the field value.
