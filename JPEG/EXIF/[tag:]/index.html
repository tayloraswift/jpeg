Subscript

[tag:]

Returns the field descriptor and boxed value for the given tag, if it exists in this metadata record.

Declaration

subscript(tag tag:UInt16) -> (type:FieldType, count:Int, box:Box)? { get }

Parameters

tag

A tag code. The various EXIF tag codes are listed in the EXIF standard. The tags dictionary contains an index of the known, defined tags in this metadata record.

Return value

The field descriptor and boxed value, if it exists, otherwise nil.
