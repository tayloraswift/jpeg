Instance Property

tags

The index of tags and fields defined in this metadata record.

Declaration

var tags:[UInt16: Int] { get }

Overview

The tag codes are listed in the EXIF standard. The dictionary values are internal file pointers that can be dereferenced manually from the storage; alternatively, the [tag:] subscript can perform both the lookup and the dereferencing in one step.
