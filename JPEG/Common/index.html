Enumeration

JPEG.Common

A built-in color format which covers the JFIF/EXIF subset of the JPEG standard.

Declaration

enum Common:JPEG.Format

Overview

This color format is able to recognize conforming JFIF and EXIF images, which use the component key assignments Y = 1, Cb = 2, Cr = 3. To provide compatibility with older, faulty JPEG codecs, it is also able to recognize non-standard component schemes as long as they have the correct arity and form a contiguously increasing sequence.
