nonconforming3x8(_:_:_:)

A non-standard 8-bit YCbCr format.

Declaration

case nonconforming3x8(JPEG.Component.Key, JPEG.Component.Key, JPEG.Component.Key)

Associated values

_

The component key interpreted as the luminance component.

_

The component key interpreted as the blue component.

_

The component key interpreted as the red component.

Overview

This color format can use any contiguously increasing sequence of component key assignments of arity 3. For example, it can use the assignments Y = 0, Cb = 1, Cr = 2, or the assignments Y = 2, Cb = 3, Cr = 4. Note that images using this format are valid JPEG images, but are not compliant JFIF or EXIF images, and some viewers may not support them.
