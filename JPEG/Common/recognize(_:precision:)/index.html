Static Method

recognize(_:precision:)

Detects this color format, given a set of component keys and a bit-depth.

Implements requirement in JPEG.Format

Declaration

static func recognize(_ components:Set<JPEG.Component.Key>, precision:Int) -> Self?

Parameters

components

Must be a numerically-contiguous set with one or three elements, or this constructor will return nil.

precision

Must be 8, or this constructor will return nil.

Overview

If this constructor detects a nonconforming3x8(_:_:_:) color format, it will populate the associated values with the keys in ascending order.
