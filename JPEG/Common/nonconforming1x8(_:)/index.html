Enumeration Case

nonconforming1x8(_:)

A non-standard 8-bit grayscale format.

Declaration

case nonconforming1x8(JPEG.Component.Key)

Associated values

_

The component key interpreted as the luminance component.

Overview

This color format can use any component key assignment of arity 1. Note that images using this format are valid JPEG images, but are not compliant JFIF or EXIF images, and some viewers may not support them.
