Instance Property

rgb

This color represented in the RGB color space.

Declaration

var rgb:JPEG.RGB

Overview

This property applies the YCbCr-to-RGB conversion formula defined in the JFIF standard. Some YCbCr colors are not representable in the RGB color space; such colors will be clipped to the acceptable range.
