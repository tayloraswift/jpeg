Enumeration Case

invalidCompositeBlockRun(_:expected:)

The decoder decoded an out-of-range end-of-band/end-of-block run count.

Declaration

case invalidCompositeBlockRun(Int16, expected:ClosedRange<Int>)

Associated values

_

The decoded end-of-band/end-of-block run count.

expected

The expected range for the end-of-band/end-of-block run count.

Overview

This error occurs when a sequential scan tries to encode an end-of-band run, which is a progressive coding process concept only. Sequential scans can only end-of-block runs of length 1.
