Enumeration Case

invalidRestartPhase(_:expected:)

A restart marker appeared out-of-phase.

Declaration

case invalidRestartPhase(Int, expected:Int)

Associated values

_

The phase read from the restart marker.

expected

The expected phase, which is one greater than the phase of the last-encountered restart marker (modulo 8), or 0 if this is the first restart marker in the entropy-coded segment.

Overview

Restart markers should cycle from 0 to 7, in that order.
