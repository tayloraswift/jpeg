Enumeration

JPEG.DecodingError

A decoding error.

Declaration

enum DecodingError:JPEG.Error

Topics

Enumeration cases

case duplicateFrameHeaderSegment

The decoder encountered more than one frame header segment.

case duplicateStartOfImage

The decoder encountered more than one start-of-image marker.

case invalidCompositeBlockRun(Int16, expected:ClosedRange<Int>)

The decoder decoded an out-of-range end-of-band/end-of-block run count.

case invalidCompositeValue(Int16, expected:ClosedRange<Int>)

The decoder decoded an out-of-range composite value.

case invalidRestartPhase(Int, expected:Int)

A restart marker appeared out-of-phase.

case invalidScanQuantizationPrecision(JPEG.Table.Quantization.Precision)

A quantization table had the wrong precision mode for the image color format.

case invalidScanSamplingVolume(Int)

An interleaved scan had a total component sampling volume greater than 10.

case invalidSpectralSelectionProgression(Range<Int>, JPEG.Component.Key)

The first scan for a component encoded a frequency band that did not include the DC coefficient.

case invalidSuccessiveApproximationProgression(Range<Int>, Int, z:Int, JPEG.Component.Key)

A scan did not follow the correct successive approximation sequence for at least one frequency coefficient.

case missingHeightRedefinitionSegment

The decoder did not encounter the height redefinition segment that must follow the first scan of an image with a declared height of 0.

case missingRestartIntervalSegment

A restart marker appeared, but no restart interval was ever defined, or restart markers were disabled.

case missingStartOfImage(JPEG.Marker)

The first marker segment in the image was not a start-of-image marker.

case prematureEndOfImage

The decoder encountered an end-of-image marker before encountering a frame header segment.

case prematureHeightRedefinitionSegment

The decoder encountered a height redefinition segment before the first image scan.

case prematureScanHeaderSegment

The decoder encountered a scan header segment before a frame header segment.

case truncatedEntropyCodedSegment

An entropy-coded segment contained less than the expected amount of data.

case undefinedScanComponentReference(JPEG.Component.Key, Set<JPEG.Component.Key>)

A scan encoded a component with a key that was not one of the resident components declared in the frame header.

case undefinedScanHuffmanACReference(JPEG.Table.HuffmanAC.Selector)

An AC huffman table selector in a scan referenced a table slot with no bound table.

case undefinedScanHuffmanDCReference(JPEG.Table.HuffmanDC.Selector)

A DC huffman table selector in a scan referenced a table slot with no bound table.

case undefinedScanQuantizationReference(JPEG.Table.Quantization.Selector)

A quantization table selector in the first scan for a particular component referenced a table slot with no bound table.

case unexpectedHeightRedefinitionSegment

The decoder encountered a height redefinition segment after, but not immediately after the first image scan.

case unexpectedRestart

The decoder encountered a restart marker outside of an entropy-coded segment.

case unrecognizedColorFormat(Set<JPEG.Component.Key>, Int, Any.Type)

A recognize(_:precision:) implementation failed to recognize the component set and bit precision in a frame header.

case unsupportedFrameCodingProcess(JPEG.Process)

The image coding process was anything other than baseline, or extended(coding:differential:) and progressive(coding:differential:) with huffman coding and differential set to false.
