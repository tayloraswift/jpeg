Enumeration

JPEG.ParsingError

A parsing error.

Declaration

enum ParsingError:JPEG.Error

Topics

Enumeration cases

case duplicateFrameComponentIndex(JPEG.Component.Key)

The same component key occurred more than once in the same frame header.

case extraneousMarkerSegmentData(JPEG.Marker, Int, expected:Int)

A marker segment contained more than the expected amount of data.

case invalidEXIFEndiannessCode((UInt8, UInt8, UInt8, UInt8))

An EXIF segment had an invalid endianness specifier.

case invalidEXIFSignature([UInt8])

An EXIF segment had an invalid signature.

case invalidFrameComponentCount(Int, JPEG.Process)

A frame header segment had an invalid number of components.

case invalidFrameComponentSamplingFactor((x:Int, y:Int), JPEG.Component.Key)

A component in a frame header had an invalid sampling factor.

case invalidFramePrecision(Int, JPEG.Process)

A frame header segment had an invalid precision field.

case invalidFrameQuantizationSelector(JPEG.Table.Quantization.Selector, JPEG.Process)

A component in a frame header segment used a quantization table selector which is well-formed but unavailable given the frame header coding process.

case invalidFrameQuantizationSelectorCode(UInt8)

A component in a frame header segment had an invalid quantization table selector code.

case invalidFrameWidth(Int)

A frame header segment had a negative or zero width field.

case invalidHuffmanTable

A huffman table definition did not define a valid binary tree.

case invalidHuffmanTargetCode(UInt8)

A huffman table definition had an invalid huffman table selector code.

case invalidHuffmanTypeCode(UInt8)

A huffman table definition had an invalid type indicator code.

case invalidJFIFDensityUnitCode(UInt8)

A JFIF segment had an invalid density unit code.

case invalidJFIFSignature([UInt8])

A JFIF segment had an invalid signature.

case invalidJFIFVersionCode((major:UInt8, minor:UInt8))

A JFIF segment had an invalid version code.

case invalidQuantizationPrecisionCode(UInt8)

A quantization table definition had an invalid precision indicator code.

case invalidQuantizationTargetCode(UInt8)

A quantization table definition had an invalid quantization table selector code.

case invalidScanComponentCount(Int, JPEG.Process)

A scan header had more that the maximum allowed number of components given the image coding process.

case invalidScanHuffmanACSelector(JPEG.Table.HuffmanAC.Selector, JPEG.Process)

A component in a frame header segment used an AC huffman table selector which is well-formed but unavailable given the frame header coding process.

case invalidScanHuffmanDCSelector(JPEG.Table.HuffmanDC.Selector, JPEG.Process)

A component in a frame header segment used a DC huffman table selector which is well-formed but unavailable given the frame header coding process.

case invalidScanHuffmanSelectorCode(UInt8)

A component in a frame header segment had an invalid quantization table selector code.

case invalidScanProgressiveSubset(band:(Int, Int), bits:(Int, Int), JPEG.Process)

A scan header specified an invalid progressive frequency band or bit range given the image coding process.

case truncatedMarkerSegmentBody(JPEG.Marker, Int, expected:ClosedRange<Int>)

A marker segment contained less than the expected amount of data.
