Instance Method

serialize(_:)

Serializes the given quantization tables as segment data.

Declaration

func serialize(_:[Quantization]) -> [UInt8]

Parameters

_

The quantization tables to serialize. The tables will appear in the serialized segment in the same order they appear in this array.

Return value

A marker segment body. This array does not include the marker type indicator, or the marker segment length field.

Overview

This method is the inverse of parse(quantization:).
