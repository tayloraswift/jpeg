Subscript

[k:h:]

Accesses the quantum value at the given grid index.

Declaration

@inlinable subscript(k k:Int, h h:Int) -> UInt16 { get set }

Parameters

k

The horizontal frequency index. This value must be in the range 0 ..< 8.

h

The vertical frequency index. This value must be in the range 0 ..< 8.

Return value

The quantum value.

Overview

Using this subscript is equivalent to using [z:] with the output of z(k:h:).
