Initializer

init(precision:values:target:)

Creates a quantization table from the given quantum values.

Declaration

init(precision:Precision, values:[UInt16], target:Selector)

Parameters

precision

The bit width of the integer type to encode the quanta as.

values

The quantum values, in zigzag order. This array must have exactly 64 elements. If the precision is uint8, all of the values must be within the range of a UInt8. Passing an invalid array will result in a precondition failure.

target

The table selector this quantization table is meant to be stored at.
