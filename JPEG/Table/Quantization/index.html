Structure

JPEG.Table.Quantization

A quantization table.

Declaration

struct Quantization:JPEG.AnyTable

Overview

Quantization tables store 64 coefficient quanta. The quantum values can be accessed using either a zigzag index with the [z:] subscript, or grid indices with the [k:h:] subscript.
