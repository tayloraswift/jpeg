z(k:h:)

Converts a coefficient grid index to a zigzag index.

Declaration

Parameters

x

The horizontal frequency index.

y

The vertical frequency index.

Return value

The corresponding zigzag index.

Overview

It is easier to convert grid indices (k, h) to zigzag indices (z) than the other way around, so most library APIs store coefficient-related information natively in zigzag order.

The JPEG format only uses the grid domain 0 ..< 8 × 0 ..< 8, which maps to the zigzag range 0 ..< 64. However, this function works for any non-negative input coordinate.
