Initializer

init(_:target:)

Creates a huffman tree from the given leaf nodes.

Declaration

init?(_ symbols:[[Symbol]], target:Selector)

Parameters

symbols

The leaf nodes in each level of the tree. The tree root is always assumed to be internal, so the 0th sub-array of this array should contain the leaves in the first level of the tree. This array must contain 16 sub-arrays, even if the deeper levels of the tree are empty, or this initializer will suffer a precondition failure.

target

The table selector this huffman table is meant to be stored at.

Overview

This initializer determines the shape of the tree from the shape of the leaf array input. It has no knowledge of symbol frequencies or priority. To build an optimal huffman tree, use the init(frequencies:target:) initializer.

This initializer will return nil if the sizes of the given leaf arrays do not describe a full binary tree. (The last level is allowed to be incomplete.) For example, the leaf counts (3, 0, 0, … ) are invalid because no binary tree can have three leaf nodes in its first level.
