Initializer

init(frequencies:target:)

Creates a huffman table containing a near-optimal huffman tree from the given symbol frequencies and table selector.

Declaration

init(frequencies:[Int], target:Selector)

Parameters

frequencies

An array of symbol frequencies. This array must contain exactly 256 elements, corresponding to the 256 possible 8-bit symbols. The ith array element specifies the frequency of the symbol with the value i.

At least one symbol must have a non-zero frequency. Passing an invalid frequency array will result in a precondition failure.

target

The selector target for the created huffman table.

Overview

This initializer uses the standard huffman tree construction algorithm to determine optimal codeword assignments. These assignments are modified slightly to fit codeword length constraints imposed by the JPEG specification.
