Instance Method

serialize(_:_:)

Serializes the given huffman tables as segment data.

Declaration

func serialize(_:[HuffmanDC], _:[HuffmanAC]) -> [UInt8]

Parameters

_

The DC huffman tables to serialize. The tables will appear in the serialized segment in the same order they appear in this array.

_

The AC huffman tables to serialize. The tables will appear in the serialized segment in the same order they appear in this array.

Return value

A marker segment body. This array does not include the marker type indicator, or the marker segment length field.

Overview

The DC tables appear before the AC tables in the serialized segment.

This method is the inverse of parse(huffman:).
