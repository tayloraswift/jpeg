JPEG.Component.Key

A unique identifier assigned to each color component in an image.

Declaration

struct Key:Hashable, Comparable, ExpressibleByIntegerLiteral

Overview

Component keys are numeric values ranging from 0 to 255. In these reference pages, component keys in their numerical representation are written in boldface.
