Generic Structure

JPEG.Layout

A specification of the components, coding process, table assignments, and scan progression of a JPEG file.

Declaration

struct Layout<Format> where Format:JPEG.Format

Overview

This structure records both the recognized components and the resident components in an image. We draw this distinction because the planes property is allowed to include definitions for components that are not part of format.components. Such components will not recieve a plane in the JPEG.Data types, but will be ignored by the scan decoder without errors.

Non-recognized components can only occur in images decoded from JPEG files, and only when using a custom JPEG.Format type, as the built-in JPEG.Common color format will never accept any component declaration in a frame header that it does not also recognize. When encoding images to JPEG files, all declared resident components must also be recognized components.
