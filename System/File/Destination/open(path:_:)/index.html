open(path:_:)

Calls a closure with an interface for writing to the specified file.

Declaration

static func open<R>(path:String, _ body:(inout Self) throws -> R) rethrows -> R?

Parameters

path

The path to the file to open.

body

A closure with a Destination parameter representing the specified file to which data can be written to. This interface is only valid for the duration of the methodâ€™s execution. The closure is only executed if the specified file could be successfully opened, otherwise this method will return nil. If body has a return value and the specified file could be opened, this method returns the return value of the closure.

Return value

The return value of the closure argument, or nil if the specified file could not be opened.

Overview

This method automatically closes the file when its closure argument returns.
